Class: FlexMultiTrafficLight

trafficLight.FlexMultiTrafficLight(trafficLights)

A composite traffic light with a flexible way to select which composed traffic lights are active or in use.

Constructor

new FlexMultiTrafficLight(trafficLights)

Creates a new instance of this class. Starts off using the first traffic light in the provided `trafficLights`. Tries to checks out the provided traffic lights.

near()

Selects the nearest traffic light to use, remembering the direction of movement (forwards or backwards). Also works with multiple selected traffic lights, moving all to the nearest, following a single direction (so it's possible to wrap around at the last if both the first and last indexes are in use).
